Project Description:
Making Smart contracts for lottery using solidity

There are two stake holders in this Process
1. Manager : Manager will be the one who will manage this game he will be able to see number of participents, no. of lottries bought, balance collection, selecting a winner, paying the winner price.

2. participents : must have walletts to do do transactions, can buy multiple lottries by sending ethers.

Functionalities --

specifying the min. amount to be paid by participents to buy a lottery, If not transaction will be failed

setting the Manager

keeping tabs of participents

seeing the total amount collected(Only by Manager)

setting minimum no of participents to start the lottery

selecting a random winner

give the winning amount to winner
